##Usage
show a simple dialog in activity

* loading

	SimpleHUD.showLoadingMessage(this, "正在加载");
	
* info

	SimpleHUD.showInfoMessage(this, "删除成功");

* success

	SimpleHUD.showSuccessMessage(this, "上传成功");
	
* error

	SimpleHUD.showErrorMessage(this, "数据异常，具体原因是什么我也不知道，哈哈")
